# TM1 CubeMap

Interactive data lineage diagram for TM1 / Planning Analytics on-prem (V11 and V12).

> **Phase 1 — Early Release.** Core features are working but you may encounter rough edges. Feedback welcome via [GitHub Issues](https://github.com/falconbi/tm1_cubemap/issues).

---

## Quick Start

```bash
mkdir tm1-cubemap && cd tm1-cubemap
curl -sSLO https://raw.githubusercontent.com/falconbi/tm1_cubemap/main/docker-compose.yml
docker compose up -d
```

Open **http://localhost:8084** — a setup form walks you through connecting to your TM1 server.
